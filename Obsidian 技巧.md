# Obsidian 技巧

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+F` | 全文搜索 |
| `Ctrl+Tab` | 切换笔记 |
| `Ctrl+E` | 切换编辑/预览模式 |
| `Ctrl+Shift+E` | 导出 PDF |

## Markdown 扩展

- `[^链接]` 可以写脚注
- `> [!note]` / `> [!warning]` 提示块
- 三个 \`\`\` 代码块
- 用 `- [ ]` 做任务列表

## 常用插件推荐

- **Obsidian Git** — 自动 commit/push
- **Dataview** — 用 SQL 查询笔记
- **Calendar** — 日记视图
